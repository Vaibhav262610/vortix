## ğŸ› Troubleshooting

### Shortcuts Not Working?

**Issue**: Ctrl+Enter doesn't send command **Solution**: Make sure:

- Device is selected
- Command input has text
- Not currently executing a command

**Issue**: Ctrl+K doesn't open modal **Solution**: Make sure:

- You're on the dashboard page
- No other modal is open
- Browser isn't blocking the shortcut

**Issue**: ESC doesn't close modal **Solution**: Make sure:

- A modal is actually open
- Browser focus is on the page
- Not in an input field (ESC may clear input first)
